# contracts/ (Normative)

This directory contains the **normative OKHAM contract artifacts** for this application.

Policy (template)
- Canonical format: **JSON** (`*.json`)
- IDs should be stable and start with the app `namespace` from `manifest/app.json`.

Structure
- `otc/types/` — data types (OTC)
- `oec/events/` — immutable facts/events (OEC)
- `oic/intents/` — intents/interactions (OIC)
- `opc/policies/` — policies/constraints (OPC)
- `oucc/ui/` — UI component contracts (OUCC)
- `onc/navigation/` — navigation/routes (ONC)
- `occ/capabilities/` — capability/function metadata (OCC/OFR)
