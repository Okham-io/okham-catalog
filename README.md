# okham-catalog

Catalogs for the OKHAM ecosystem (product org).

This repo is intended to host **runtime-agnostic OKHAM app packages**, component catalogs, and templates.

- Standards/specs live at https://okham.org and https://github.com/Okham-org/okham-specs
- Tooling lives at https://github.com/Okham-io/okham-tools

## Repository layout (v0)

- `templates/okham-app/` — a minimal OKHAM app package template
- `packages/` — concrete app packages (runtime-agnostic contracts + optional bindings)
